# Goals

2 player (WASD + arrow keys) roll-a-ball game

# Play

Playable in web browser at https://Veaizy.github.io/RollABall/ .

# Paid dependencies

The following paid assets are needed to run this project:
1. Adaptive Split Screen - https://lukaschod.github.io/adaptive-split-screen-docs/manual/index.html
2. ShareDevice - https://assetstore.unity.com/packages/tools/input-management/share-inputsystem-device-300977
